Universal Selector

  • Syntax : 
                            * { 
                            style properties 
                        }

  • CSS code : 
    •                             *{
                                color: forestgreen;
                                font: "22px";
                            }
                            .p1 {
                                color: red;
                            }
                            .p2 {
                                color: red;
                            }

  • Example :

    • Universally i declare green color to all text.

    and hear i declare class list on < ol >, so list will appear in green color and other texts are appear as a red color.
