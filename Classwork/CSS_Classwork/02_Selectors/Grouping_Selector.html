Grouping Selector

  • Syntax : 
    
                        element, element  { 
                            style properties 
                        }

  • CSS code : 

    
                    * {
                        font-size: 22px;
                    }
                    
                    p,h3 {
                        color: red;
                        background-color: lightyellow;
                        
                    }
