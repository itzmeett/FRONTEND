Font
  • In CSS there are five generic font families:

    1. Serif fonts have a small stroke at the edges of each letter.
    2. Sans-serif fonts have clean lines (no small strokes attached).
    3. Monospace fonts - here all the letters have the same fixed width.
    4. Cursive fonts imitate human handwriting.
    5. Fantasy fonts are decorative/playful fonts.

  • Example :

    This is belongs to "Times New Roman", serif font-family

    This is belongs to "Arial", san-serif font-family

    This is belongs to "Lucida Console", "Courier New", monospace font-family







Font Style
  • This property has three values:

    1. normal - The text is shown normally
    2. italic - The text is shown in italics
    3. oblique - The text is "leaning" (oblique is very similar to italic, but less supported)

    4. Example :

      This is a paragraph in normal style.

      This is a paragraph in italic style.

      This is a paragraph in oblique style.

    5. Code : 
                                  

                              p.normal {
                                    font-style: normal;
                              }
                              
                              p.italic {
                                font-style: italic;
                              }
                              
                              p.oblique {
                                  font-style: oblique;
                              }

Font Weight
  • The font-weight property specifies the weight of a font:

    • Example :

      • This is a normal paragraph.

      This is bold paragraph


    • Code : 
      •                             
                                .bold {
                                    font-weight: bold;
                                }
