Selectors
  • A CSS selector selects the HTML element you want to style.

01 Element Selector
  • The element selector selects HTML elements based on the element name.

  • Example :
    •     p {
         text-align: center;
         color: red;
        }

02 ID Selector
  • The id selector uses the id attribute of an HTML element to select a specific element.
  • The id of an element is unique within a page, so the id selector is used to select one unique element!
  • To select an element with a specific id, write a hash (#) character, followed by the id of the element.
  • The CSS rule below will be applied to the HTML element with id="para1" :

  • Example :
    •     #para1 {
         text-align: center;
         color: red;
        }

03 Class Selector
  • The class selector selects HTML elements with a specific class attribute.
  • To select elements with a specific class, write a period (.) character, followed by the class name.

  • Example - 1 :

        .center {
         text-align: center;
         color: red;
        }

  • Example - 2 :

        p.center {
         text-align: center;
         color: red;
        }

  • This <p> element will be styled according to class="center" AND to class="large":

    Example - 3 :

        <p class="center large">This paragraph refers to two classes.
        </p>

04 Universal Selector
  • The universal selector (*) selects all HTML elements on the page.
  • The CSS rule below will affect every HTML element on the page :

    Example :

        * {
         text-align: center;
         color: red;
        }

05 Grouping Selector
  • The grouping selector selects all the HTML elements with the same style definitions.
  • Look at the following CSS code (the h1, h2, and p elements have the same style definitions) :

    Example :

        h1, h2, p {
         text-align: center;
         color: red;
        }

06 Descendant Selector
  • The descendant selector matches all elements that are descendants of a specified element.
  • The following example selects all <p> elements inside <div> elements:

    Example :

        div p {
         background- color : yellow;
        }

07 Child Selector
  • The child selector selects all elements that are the children of a specified element.
  • The following example selects all <a> elements that are children of a <p> element inside <div> element :

    Example :

        div>p>a {
         background- color : yellow;
        }

08 Adjacent Sibiling Selector
  • The adjacent sibling selector is used to select an element that is directly after another specific element.
  • Sibling elements must have the same parent element, and "adjacent" means "immediately following".
  • The following example selects the first <p> element that are placed immediately after <div> elements: :

    Example :

        div + a {
         background- color : yellow;
        }

09 General Sibiling Selector
  • The general sibling selector selects all elements that are next siblings of a specified element.
  • The following example selects all <p> elements that are next siblings of <p> elements:

    Example :

        div ~ p {
         background- color : yellow;
        }

Selectors

  1.  . => class

  2.  # => ID

  3.  h1,h2,p,div => ElementTypeSelectors

  4.  ul.menu a{ text-decoration: none;} / ul#menu a{ text-decoration: none;} => DescendantSelectors

  5.  .menu>li>label>a (following hyrarchi) / .menu a => ChildSelectors

  6.  .menu>li>label>a / .menu a => ChildSelectors

  7.   h2+p => AdjacentSiblingSelectors immidiate next

  8.   h2~p => GeneralSiblingSelectors next

  9.   h2,p,lable => GroupingSelectors for Multiple selectors
