2.1 Define the terms: Website, Webpage, Web browser, Web server, HTML, CSS

Website
  • A collection of web pages which are grouped together and usually connected together in various ways. Often called a "website" or a "site."

    • Webpage
  • A document which can be displayed in a web browser such as Firefox, Google Chrome, Opera, Microsoft Edge, or Apple Safari. These are also often called just "pages."

    • Web browser
  • The web browser is an application software to explore www (World Wide Web). It provides an interface between the server and the client and requests to the server for web documents and services.

    • Web server
  • A computer that hosts a website on the Internet.

    • HTML
  • HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content.

    • CSS
  • CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media
