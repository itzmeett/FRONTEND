Basics

  • WEB :

    - Whatever u want to share with other via internet that facility provided by web

  • WWW :

    - World Wide Web is a collection of websites or webpage stored in web server and connected to local computers through the internet.

  • Browser :

    - Is a software program used to explore, retrieve, and display the information available on the world wide web.

  • Addressbar :

    - Is referred to as an address box, location bar, URL. In the internet browser’s address bar is a feature to enter and display a website’s address.

  • Titlebar :

    - The title bar is a horizontal bar located at the top of a window in a GUI.

  • HTTP/HTTPS :

    - It is a protocol used to access data on the World Wide Web (www).

    - Hypertext transfer protocol secure (HTTPS) is the secure version of HTTP, which is the primary protocol used to send data between a web browser and a website. HTTPS is encrypted in order to increase security of data transfer.

  • URL/URI :

    - A uniform resource locator, that only indicates the location of a web page.

    - A uniform resource identifier (URI) syntax defines its structure, allowing a program to understand it.

  • Extension :

    - Different programming language have different extension, it is refer to respective programming language.

  • Editor :

    - Editors are software that allows user to create and edit new text file.

  • Network :

    - In information technology, a network defines connection of at least two computer systems via cable or wireless connection.

  • TCP/IP :

    - Transmission Control Protocol / Internet Protocol is communication protocols that define transmitting data over network.

  • Hosting :

    - Web hosting is a process or service through web application or website are stored in web server to publish to the internet via world wide web.

  • Domain :

    - Domain name is identified an IP address for example google.com points to the IP address "74.125.127.147". Domain name is identified to remember a name rather than IP address.

  • SERVER :

    - Is referred to computer program to receive and response to request made by computer network by user or client.

  • Client :

    - A client is a computer that connects to and uses the resources of a remote computer or server.

  • SEO :

    - Search Engine Optimization is the process of improving the visibility of a website in search engines.
